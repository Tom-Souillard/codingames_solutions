# Linked List Cycle

## 🎯 Objectif

- Déterminer si une liste chaînée contient un cycle.

---

## 📝 Détails

- Utilise l'algorithme de Floyd (Tortue et Lièvre) pour la détection de cycle.

---

## 📥 Entrée

- head: La tête de la liste chaînée.

---

## 📤 Sortie

- bool: True si un cycle est présent, False sinon.
