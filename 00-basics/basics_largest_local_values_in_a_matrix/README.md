# Largest Local Values in a Matrix

## 🎯 Objectif

    Trouver la valeur maximale dans chaque sous-matrice 3x3 d'une matrice n x n donnée.

---

## 📝 Détails

    Pour chaque sous-matrice 3x3 centrée autour de l'élément (i+1, j+1) de la matrice initiale, calculer la valeur maximale et la stocker dans une nouvelle matrice (n-2) x (n-2).

---

## 📥 Entrée

    Une matrice carrée grille de taille n x n remplie d'entiers.

---

## 📤 Sortie

    Une matrice de taille (n-2) x (n-2) où chaque élément est le maximum d'une sous-matrice 3x3 dans la grille originale.
