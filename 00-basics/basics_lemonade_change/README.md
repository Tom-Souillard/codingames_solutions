# Lemonade Change

## 🎯 Objectif

Vérifier la validité de la distribution de la monnaie.

---

## 📝 Détails

Implémente une fonction pour vérifier si chaque client reçoit la monnaie correcte.

---

## 📥 Entrée

Liste de billets utilisés par les clients pour payer.

---

## 📤 Sortie

Booléen indiquant si la monnaie peut être rendue correctement.
