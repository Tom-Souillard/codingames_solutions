# Build an Array With Stack Operations

## 🎯 Objectif

- Générer une séquence d'opérations pour construire une pile cible à partir d'une séquence d'entiers.

---

## 📝 Détails

- Utilise les opérations "Push" pour ajouter et "Pop" pour retirer des entiers.
- La construction s'arrête dès que la pile correspond à la cible.

---

## 📥 Entrée

- Utilise les opérations "Push" pour ajouter et "Pop" pour retirer des entiers.
- La construction s'arrête dès que la pile correspond à la cible.

---

## 📤 Sortie

- Liste de chaînes représentant les opérations pour atteindre la cible (List[str]).
