# House Robber

## 🎯 Objectif

- Calculer le montant maximum d'argent pouvant être volé sans alerter la police en ne volant pas dans des maisons adjacentes.

---

## 📝 Détails

- Utilisation de la programmation dynamique.
- Complexité temporelle : O(n), où n est le nombre de maisons.
- Complexité spatiale : O(1).
---

## 📥 Entrée

- `maisons` (List[int]) : Liste d'entiers représentant l'argent dans chaque maison.

---

## 📤 Sortie

- `int` : Montant maximum d'argent qui peut être volé.
